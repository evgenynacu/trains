api

  запрос на получение расписание между станциями

интерфейс
  поле с поиском
  окно поиск

    запоминаются последние введенные станции и расписание для них синхронизируется в фоне, в обе стороны
    2(3) вкладки: сегодня, завтра(послезавтра)
    возможность добавить в избранное
    дата последнего обновления где-то
  окно избранное
    список избранных маршрутов
    синхронизируются в фоне
    строка - маршрут + ближайшие времена отправления
    click переводит на окно поиска
    кнопка "refresh"
    дата последнего обновления где-то
